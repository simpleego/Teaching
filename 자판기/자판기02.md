## ☕ **[시나리오] 믹스 커피 자판기 2단계**

1. 사용자가 **돈을 넣는다.**  
2. 커피 메뉴를 선택한다.  
   - 1번: 블랙 커피(300원)  
   - 2번: 믹스 커피(300원)  
   - 3번: 설탕 커피(300원, 설탕량 선택 가능)  
3. 설탕 커피를 선택한 경우  
   - 설탕을 0~3 스푼 중 선택  
4. 선택한 메뉴에 따라 결과 출력  
5. 잔돈 출력  

---

# ✅ **2단계 완성 코드 (조건문 중심)**

```java
import java.util.Scanner;

public class CoffeeMachineStep2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 1. 돈 입력
        System.out.print("돈을 넣어주세요: ");
        int money = sc.nextInt();

        int price = 300;

        // 2. 메뉴 선택
        System.out.println("=== 메뉴 선택 ===");
        System.out.println("1. 블랙 커피");
        System.out.println("2. 믹스 커피");
        System.out.println("3. 설탕 커피");
        System.out.print("메뉴 번호 입력: ");
        int menu = sc.nextInt();

        String result = "";

        // 3. 조건문으로 메뉴 처리
        if (menu == 1) {
            result = "블랙 커피가 준비되었습니다.";
        } 
        else if (menu == 2) {
            result = "믹스 커피가 준비되었습니다.";
        } 
        else if (menu == 3) {
            System.out.print("설탕은 몇 스푼 넣을까요? (0~3): ");
            int sugar = sc.nextInt();

            if (sugar < 0 || sugar > 3) {
                System.out.println("설탕은 0~3 스푼만 가능합니다.");
                return;
            }

            result = "설탕 " + sugar + "스푼 커피가 준비되었습니다.";
        } 
        else {
            System.out.println("잘못된 메뉴 번호입니다.");
            return;
        }

        // 4. 잔돈 계산
        if (money < price) {
            System.out.println("돈이 부족합니다. 커피를 구매할 수 없습니다.");
            return;
        }

        int change = money - price;

        // 5. 결과 출력
        System.out.println(result);
        System.out.println("잔돈: " + change + "원");
    }
}
```

# ✅ **2단계 핵심 해설**

### ✅ 1. 조건문으로 메뉴 선택
```java
if (menu == 1) { ... }
else if (menu == 2) { ... }
else if (menu == 3) { ... }
```
- 메뉴 번호에 따라 다른 문장을 출력  
- 조건문 학습에 최적화된 구조

---

### ✅ 2. 설탕 커피는 추가 입력 필요
```java
int sugar = sc.nextInt();
```
- 설탕 스푼 수를 입력받고  
- 범위 체크(0~3)로 조건문 연습

---

### ✅ 3. 잔돈 계산
```java
int change = money - price;
```
- 1단계에서 배운 연산자 개념을 그대로 활용

---

### ✅ 4. 잘못된 입력 처리
```java
else {
    System.out.println("잘못된 메뉴 번호입니다.");
    return;
}
```
- 조건문에서 **예외 처리** 개념을 자연스럽게 학습
