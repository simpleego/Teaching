# ✅ **📘 시나리오(1단계 버전)**

학생들이 이해하기 쉬운 **아주 단순한 자판기 흐름**으로 구성했어.

---

## ☕ **[시나리오] 믹스 커피 자판기 1단계**

1. 사용자가 **돈을 넣는다.**  
2. 믹스 커피 한 잔의 가격은 **300원**이다.  
3. 사용자가 넣은 돈으로 **몇 잔의 커피를 살 수 있는지 계산**한다.  
4. 커피를 구매하고 남는 **잔돈을 계산**한다.  
5. 결과를 출력한다.

---

### ✅ **학생들이 배우는 개념**
- `int money = sc.nextInt();` → 입력  
- `int cup = money / 300;` → 산술 연산  
- `int change = money % 300;` → 나머지 연산  
- `System.out.println()` → 출력  
- 변수 선언과 저장

---

# ✅ **1단계 완성 코드 (입출력 + 변수 + 연산자)**

```java
import java.util.Scanner;

public class CoffeeMachineStep1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 1. 돈 입력받기
        System.out.print("돈을 넣어주세요: ");
        int money = sc.nextInt();

        // 2. 커피 가격
        int price = 300;

        // 3. 구매 가능한 커피 잔 수 계산
        int cup = money / price;

        // 4. 잔돈 계산
        int change = money % price;

        // 5. 결과 출력
        System.out.println("구매 가능한 커피: " + cup + "잔");
        System.out.println("잔돈: " + change + "원");
    }
}
```

---

# ✅ **코드 해설(학생용)**

| 코드 요소 | 설명 |
|----------|------|
| `Scanner` | 사용자 입력을 받기 위한 도구 |
| `int money` | 사용자가 넣은 돈을 저장하는 변수 |
| `int price = 300` | 커피 한 잔 가격 |
| `money / price` | 몇 잔을 살 수 있는지 계산 |
| `money % price` | 잔돈 계산 |
| `System.out.println()` | 결과 출력 |
