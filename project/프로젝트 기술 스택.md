# 프로젝트 기술 스택

**스프링부트 기반 프로젝트에서 활용 가능한 기술 스택은 백엔드, 프론트엔드, 데이터베이스, 인프라, 개발 도구 등으로 구성됩니다. 아래에 각 영역별로 주요 기술들을 정리했습니다.**

---

### 🧠 백엔드 기술 스택 (Spring Boot 중심)
- **프레임워크**
  - *Spring Boot*: 핵심 백엔드 프레임워크
  - *Spring MVC*: 웹 애플리케이션 구조화
  - *Spring Security*: 인증 및 권한 관리
  - *Spring Data JPA*: ORM 기반 데이터 접근
  - *Spring Batch*: 대용량 배치 처리
  - *Spring Cloud*: 마이크로서비스 아키텍처 지원

- **언어 및 빌드 도구**
  - *Java*, *Kotlin*
  - *Gradle*, *Maven*

- **API 개발**
  - *RESTful API*, *GraphQL*
  - *Swagger/OpenAPI*: API 문서 자동화

---

### 🎨 프론트엔드 기술 스택
- **템플릿 엔진**
  - *Thymeleaf*, *JSP*, *Freemarker*

- **SPA 프레임워크 (프론트 분리형)**
  - *React*, *Vue.js*, *Angular*

- **통신**
  - *Axios*, *Fetch API*, *WebSocket*

---

### 🗄️ 데이터베이스 및 캐시
- **관계형 DB**
  - *MySQL*, *PostgreSQL*, *MariaDB*, *Oracle*

- **NoSQL**
  - *MongoDB*, *Redis*, *Elasticsearch*

- **ORM**
  - *JPA (Hibernate)*, *QueryDSL*

- **캐시**
  - *Redis*, *Ehcache*, *Caffeine*

---

### ☁️ 인프라 및 배포
- **서버 및 클라우드**
  - *AWS (EC2, RDS, S3)*, *Azure*, *GCP*
  - *Nginx*, *Apache*

- **CI/CD**
  - *Jenkins*, *GitHub Actions*, *GitLab CI*, *CircleCI*

- **컨테이너 및 오케스트레이션**
  - *Docker*, *Kubernetes*, *Helm*

- **모니터링 및 로깅**
  - *Prometheus*, *Grafana*, *ELK Stack (Elasticsearch, Logstash, Kibana)*

---

### 🛠️ 개발 및 협업 도구
- **IDE**
  - *IntelliJ IDEA*, *VS Code*, *Eclipse*

- **버전 관리**
  - *Git*, *GitHub*, *GitLab*, *Bitbucket*

- **테스트**
  - *JUnit*, *Mockito*, *RestAssured*, *Testcontainers*

- **문서화 및 협업**
  - *Swagger*, *Confluence*, *Jira*, *Notion*

---

### 🔐 인증 및 보안
- **OAuth2 / JWT 기반 인증**
  - *Spring Security + OAuth2*, *Keycloak*, *Auth0*

- **암호화 및 보안**
  - *HTTPS*, *SSL 인증서*, *Spring Security 암호화 모듈*

---

이 기술 스택은 프로젝트의 규모, 목적, 팀 구성에 따라 선택적으로 조합됩니다. 예를 들어, 스타트업에서는 React + Spring Boot + MySQL + AWS 조합이 흔하고, 대기업에서는 Spring Cloud + Kubernetes + Oracle + ELK Stack을 활용하는 경우가 많습니다.

원하는 프로젝트 유형이나 규모에 따라 맞춤형 스택을 추천해줄 수도 있어요. 어떤 프로젝트를 계획 중이신가요?
