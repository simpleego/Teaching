### AWS 인스턴스를 설정하여 접속할 환경을 SSH로 설정한다.
> 접속에 필요한 정보를 화면에서 얻을 수 있다.

```
scp -i "키페어.pem" 보낼파일경로 사용자계정@ec2아이피:받을경로





java -version
sudo apt update -- apt를 최신버전으로 업데이트 후 설치를 진행한다.
java -version -- 설치가능한 자바 버전을 다시 확인
sudo apt install openjdk-17-jre-headless  -- openjdk-17를 설치한다.

nohup java -jar file-0.0.1-SNAPSHOT.jar &
```
