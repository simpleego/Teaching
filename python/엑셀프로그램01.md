파이썬으로 엑셀을 다루는 가장 실용적인 예제 5가지를 소개해드리겠습니다.

## 1. **엑셀 파일 읽기 및 데이터 추출**
```python
import pandas as pd

# 엑셀 파일 읽기
df = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# 특정 열 데이터 추출
names = df['이름']
sales = df['매출']

# 조건에 맞는 데이터 필터링
high_sales = df[df['매출'] > 1000000]
print(high_sales)
```

## 2. **여러 엑셀 파일 병합**
```python
import pandas as pd
import glob

# 같은 폴더의 모든 엑셀 파일 찾기
all_files = glob.glob('*.xlsx')

# 모든 파일을 읽어서 하나로 합치기
df_list = []
for file in all_files:
    df = pd.read_excel(file)
    df_list.append(df)

# 병합
merged_df = pd.concat(df_list, ignore_index=True)
merged_df.to_excel('merged_result.xlsx', index=False)
```

## 3. **데이터 집계 및 피벗 테이블**
```python
import pandas as pd

df = pd.read_excel('sales_data.xlsx')

# 부서별 매출 합계
dept_sales = df.groupby('부서')['매출'].sum()

# 피벗 테이블 생성
pivot = pd.pivot_table(df, 
                       values='매출', 
                       index='부서', 
                       columns='월',
                       aggfunc='sum')

pivot.to_excel('pivot_result.xlsx')
```

## 4. **엑셀에 차트가 포함된 보고서 생성**
```python
import pandas as pd
from openpyxl import load_workbook
from openpyxl.chart import BarChart, Reference

# 데이터 작성
df = pd.DataFrame({
    '제품': ['A', 'B', 'C', 'D'],
    '매출': [1000, 1500, 1200, 1800]
})
df.to_excel('report.xlsx', index=False, sheet_name='매출')

# 차트 추가
wb = load_workbook('report.xlsx')
ws = wb['매출']

# 차트 생성
chart = BarChart()
data = Reference(ws, min_col=2, min_row=1, max_row=5)
categories = Reference(ws, min_col=1, min_row=2, max_row=5)

chart.add_data(data, titles_from_data=True)
chart.set_categories(categories)
chart.title = "제품별 매출"

ws.add_chart(chart, "D2")
wb.save('report.xlsx')
```

## 5. **엑셀 자동화: 여러 시트에 데이터 분할 저장**
```python
import pandas as pd

# 대용량 데이터 읽기
df = pd.read_excel('all_data.xlsx')

# ExcelWriter로 여러 시트에 저장
with pd.ExcelWriter('분류된_데이터.xlsx', engine='openpyxl') as writer:
    # 부서별로 분할하여 각각 다른 시트에 저장
    for dept in df['부서'].unique():
        dept_df = df[df['부서'] == dept]
        dept_df.to_excel(writer, sheet_name=dept, index=False)
    
    # 요약 시트 추가
    summary = df.groupby('부서')['매출'].sum()
    summary.to_excel(writer, sheet_name='요약')

print("완료!")
```
