- 부제: “쉽고 유연한 온라인 학습 환경 만들기”
- 발표자 이름 / 날짜

### 📌 슬라이드 2: 프로젝트 개요
- 왜 LMS가 필요한가?
  - 온라인 수업 증가
  - 학습자와 교수자 간의 소통 필요
- 프로젝트 목표
  - 누구나 쉽게 사용할 수 있는 학습 플랫폼 개발

---

### 👥 슬라이드 3: 사용자와 기능
- 사용자 유형
  - 👨‍🏫 교수자: 강의 등록, 과제 관리
  - 👩‍🎓 학습자: 수강 신청, 과제 제출, 성적 확인
  - 🛠️ 관리자: 사용자 관리, 시스템 설정
- 주요 기능
  - 강의 관리, 출석 체크, 과제/시험, 커뮤니티

### 🧠 슬라이드 4: 시스템 구조 (쉬운 설명)
- 웹 기반 시스템
- 프론트엔드: 사용자가 보는 화면 (예: 홈페이지)
- 백엔드: 데이터를 처리하는 엔진 (예: 성적 계산)
- 데이터베이스: 정보를 저장하는 공간 (예: 학생 목록)

👉 그림으로 간단한 구조도 삽입 추천
