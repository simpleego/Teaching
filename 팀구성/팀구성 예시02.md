# ✅ **예시 4 — 학생 D**

## 1) 사전면담결과

* 프로젝트 리더 역할 경험이 있으며 일정 관리를 체계적으로 진행하는 스타일.
* 대화가 명확하고 팀 내 조율 능력이 뛰어남.
* 다만, 완성도에 대한 기준이 높아 팀원들에게 일정 수준을 요구하는 경향이 있음.

## 2) 학생 역량분석

* Spring MVC 구조와 API 설계 능력이 뛰어나며 MyBatis 매퍼 구성에 익숙함.
* 서버 로직 구성을 빠르게 수행하고 문제 해결 능력이 우수함.
* 프로젝트 구조 설계나 전체적인 아키텍처 고민을 주도할 수 있음.

## 3) 선호분야 조사 결과

* **희망 역할:** 팀장 + 전체 아키텍처 설계 + 백엔드
* **선호 기술:** Spring, MyBatis, REST API
* **기피 분야:** 세부 UI 스타일링, CSS
  → **배치 근거:** PM 및 백엔드 핵심 로직 담당자로 적합

# ✅ **예시 5 — 학생 E**

## 1) 사전면담결과

* 성격이 온화하여 팀 분위기 형성에 긍정적으로 기여하는 유형.
* 협업 중 의견 충돌 시 상대방 의견을 먼저 듣고 조율하는 태도를 보임.
* 종종 의사결정이 느린 편이지만 주어진 업무는 끝까지 책임감 있게 수행함.

## 2) 학생 역량분석

* HTML/CSS 구조 이해도가 높고 반응형 UI 구현에 안정적인 편임.
* React 기본기의 숙련도는 보통이지만 꾸준히 보완하는 자세가 긍정적.
* 테스트 수행 능력이 좋으며 오류 발견에 강점이 있음.

## 3) 선호분야 조사 결과

* **희망 역할:** 프론트엔드 보조, UI 마크업
* **선호 기술:** HTML/CSS, React 기본
* **기피 분야:** 복잡한 백엔드 로직
  → **배치 근거:** 화면 구성 및 UX 디테일 조정에 적합

# ✅ **예시 6 — 학생 F**

## 1) 사전면담결과

* 학습 속도가 빠르고 새로운 기술을 시도하는 데 거부감이 없음.
* 상대적으로 혼자 집중해서 개발하는 스타일이라 팀 내 소통이 조금 부족할 수 있음.
* 기술적인 도전 의지가 강해 난이도 있는 기능 구현을 맡고 싶어함.

## 2) 학생 역량분석

* React 고급 문법(Hooks, Custom Hook) 및 라이브러리 사용 경험 보유.
* 서버-클라이언트 연동 능력이 뛰어나고 API 사용 능력 우수.
* 개발 속도가 매우 빠르나 간혹 코드 품질이 떨어지는 경우 있어 코드 리뷰 필요.

## 3) 선호분야 조사 결과

* **희망 역할:** 프론트엔드 메인 개발
* **선호 기술:** React, Redux, Axios
* **기피 분야:** 기획/문서화
  → **배치 근거:** 프론트 핵심 기능 개발 담당으로 적합

* 테스트 케이스 구성 능력도 뛰어나 QA 가능.

## 3) 선호분야 조사 결과

* **희망 역할:** 백엔드 + QA
* **선호 기술:** MyBatis, SQL, Spring
* **기피 분야:** UI 구현
  → **배치 근거:** DB 중심 기능 및 QA 담당으로 적합
